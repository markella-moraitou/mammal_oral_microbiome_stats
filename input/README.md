## General ##

**decOM_output.csv** : k-mer based sourcetracking results

**host_taxonomy.csv** : Taxonomic info on mammalian host species

**Lintulaakso_diet_filtered.csv** : Dietary information for the host species, subsetted from Lintulaakso et al. database

**mammal_vertlife.nex** : Mammalian host phylogenetic tree from vertlife

**read count.csv** : Read count for all samples throughout all pre-processing steps

**read length.csv** : Average read length for all samples throughout all pre-processing steps

**sample_metadata.csv** : Collection and laboratory metadata for all samples


## For taxonomic analysis ##

**malt_abundance_matrix_sam.txt** : Abundance table output from aMeta (MIGHT CHANGE)


## For functional analysis ##

**gene_abundance_stratified.tsv** : Abundance of genes annotated in DRAM based on contig average depth
